# Moonshot

## Grant Phase

We applied to Moonshot for a grant in the **15k–30k** range to:

- fund **core development**
- integrate **more rails**
- accelerate the **LATAM launch**

This is the immediate goal of the application.

## Post-Grant Collaboration Phase

If the project is selected among the top positions, we also see room for a later collaboration phase that could generate revenue for them beyond the **10% equity** already attached to the program.

This would be a separate step after the grant phase, and we prefer to state it transparently.

Possible collaboration paths include:

- integrating **their stablecoin**, preferably on **Liquid** or **Polygon**
- enabling **PYG/USD cashout** on **Ueno accounts**
- supporting **semi-instant settlement** flows, similar to alias-based UX
